SCSS (Sassy CSS)- All Features
Variables in SCSS
  • Variables allow storing values (like colors, fonts, and spacing) for reuse.
  • Helps maintain consistency and makes updates easier.
    •                 $primary-color: #3498db;
                $font-size: 16px;
                body {
                    background-color: $primary-color;
                    font-size: $font-size;
                }
    Nesting in SCSS
  • Helps maintain a cleaner structure by nesting selectors within parent selectors.
  • Eliminates redundancy in selector names and improves readability.
    •                 nav {
                    ul {
                        li {
                            a {
                                color: blue;
                                text-decoration: none;
                            }
                        }
                    }
                }
    Mixins in SCSS
  • Mixins allow reusable blocks of styles, reducing duplication.
  • Can accept parameters to create dynamic styles.
    •                 @mixin border-radius($radius) {
                    border-radius: $radius;
                }
                button {
                    @include border-radius(5px);
                }
    Extend/Inheritance
  • Helps avoid code duplication by allowing styles to inherit from others.
  • Use %placeholder to define reusable styles.
    •                 %btn-base {
                    padding: 10px;
                    border: none;
                }
                .primary-btn {
                    @extend %btn-base;
                    background-color: blue;
                }
                .secondary-btn {
                    @extend %btn-base;
                    background-color: gray;
                }
    Operators in SCSS
  • SCSS allows mathematical operations like addition, subtraction, multiplication, and division.
    •                 .container {
                    width: 100% - 20px;
                    padding: 10px * 2;
                }
    Partials and Import
  • SCSS allows splitting code into smaller files called partials.
  • Partials start with an underscore (_variables.scss).
  • Import them into a main file using @import.
    •                 @import 'variables';
                @import 'mixins';
    Functions in SCSS
  • Custom functions process values dynamically.
  • Useful for calculations and reusable logic.
    •                 @function double($size) {
                    @return $size * 2;
                }
                div {
                    width: double(10px);
                }
    Loops in SCSS
  • Iterate over lists/maps to generate styles dynamically.
    •                 @for $i from 1 through 3 {
                    .item-#{$i} {
                        width: $i * 10px;
                    }
                }
    If/Else Conditions
  • SCSS allows adding logic to styles based on conditions.
    •                 $theme: dark;
                @if $theme == dark {
                    body {
                        background-color: black;
                    }
                } @else {
                    body {
                        background-color: white;
                    }
                }
    Maps in SCSS
  • SCSS maps store key-value pairs for structured data.
  • Use map-get() to retrieve values dynamically.
    •                 $colors: (primary: blue, secondary: red);
                button {
                    color: map-get($colors, primary);
                }
    Interpolation
  • Dynamically insert variable values into selectors, properties, or values.
    •                 $size: large;
                .button-#{$size} {
                    font-size: 20px;
                }
    Error Handling
  • Helps debug SCSS with warnings and errors.
    •                 @warn "This is a warning";
                @error "This is an error";
